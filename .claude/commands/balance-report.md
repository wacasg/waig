description: ゲームバランスをシミュレーションして現状レポートを出す
argument-hint: "[試行回数 例:50000]"
---

`scripts/balance_sim.py` を実行して現状のゲームバランスを評価する。

```bash
python3 scripts/balance_sim.py --games ${ARGUMENTS:-20000} --seed 1
```

出力（戦略別クリア率・平均売上・ランク分布）を読み、以下を日本語でまとめる:

1. **現状の難易度**: 「AI全追従」「最適」のクリア率が CLAUDE.md §3 の目安
   （AI全追従 45–70% / 最適 85%+）に収まっているか。
2. **問題点**: 簡単すぎ／難しすぎ／ランク偏りがあれば指摘。
3. **改善案**: あれば `target` や該当スロットの `eff` の具体的な変更案を提示
   （実際の調整は `game-balance` スキル／`balance-analyst` エージェントで前後比較しながら行う）。

数値だけ羅列せず、「高校生がプレイしてどう感じるか」に翻訳して説明すること。
