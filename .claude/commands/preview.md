description: ローカルでゲームを確認する（簡易サーバ起動＋プレイ手順）
---

`index.html` をローカルで確認する。ビルド不要。

1. 既に 8765 番が使われていないか確認し、簡易サーバをバックグラウンドで起動する:
   ```bash
   python3 -m http.server 8765
   ```
   （`run_in_background` で起動し、URL `http://localhost:8765/` を案内する）
2. 可能なら Claude_Preview MCP（`mcp__Claude_Preview__*`）または Playwright（`.mcp.json`）で
   `http://localhost:8765/` を開いてスクリーンショットを撮り、表示崩れが無いか確認する。
3. 動作チェック観点を提示する:
   - スタート → 3ターン → 結果 まで通しでプレイできる
   - 各ターンで A / B / C すべて選べる、AI相談ボタンが動く
   - ブラウザのコンソールにエラーが出ていない
   - 結果画面のランク・売上・バナー・申込ボタンが正しく出る

確認が終わったらサーバは停止してよい（不要ならポートを開けっぱなしにしない）。
